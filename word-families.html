Word Families

To understand Word Families, one needs to familiarize oneself with the Plastic Language Concept.

The Plastic Language Concept is where Words can alternate between Dynamic Filters and there respective Filters. Gerunds can alternate between the Dynamic Filter Logic Words with the Filter Gerunds, and Dynamic Filter Verbs and the Filter Gerunds. Runner can also be in the Dynamic Filter Nouns and the Filter Nouns. (Nouns>Nouns). When the word runner is in the Dynamic Filter Nouns and runner is not a Gerund.

Dynamic Filter Nouns has six Filters - Nouns, Co-Nouns, Propper Noun, Persona Noun, I Object and Adjective.

Dynamic Filter Logic Words has two Filters - Logic Words and Gerunds

Dynamic Filter Verbs has two Filters - Verbs and Gerunds.

Miscellaneous, Dynamic Filter Indicator Words are not in the Plastic Language Concept.

Indicator Words has two Filters - Portal Words and Finite Words

When a Noun is used as a Logic Word, the Noun is a Gerund. Gerunds can be Logic Words when Gerunds are in the Logic Word Dynamic Filter, meaning a Noun can be used as a Logic Word. A Noun can also be used as a Verb and Noun is called a Gerund when Noun is used as a Verb. (Logic Word>Gerund and Verb>Gerund).

As all Nouns can be used as Gerunds, we do not have a Filter Gerunds in Nouns Dynamic Filter.

Not all Logic Words can be Nouns. Only Noun Words used in Filters Gerunds can be called Logic Words or Verbs. In the example below, the words run, running, and ran are in the Word Family of runner. Runner can be a Noun, Logic Word or Verb. Runs, running and ran are Logic Words or Verbs depending on the Grammar of the sentence.

Logic Words can be Verbs. This is a part of the Plastic Language Concept.

All Logic Words can be used as Verbs. Not all Logic Words can be used as Nouns. Logic Words are Experience Words, and the Experience of a Logic Word can be projected as a Verb. Verbs being Descriptive Words in the Language Manual. 

Runner as Logic Word

runner     — Logic Word>Gerund
runs         — Logic Word>Logic Word
run           — Logic Word>Logic Word
running   — Logic Word>Logic Word
ran           — Logic Word>Logic Word


Runner as Verb

runner     — Verb>Gerund
runs         — Verb>Verb
run           — Verb>Verb
running   — Verb>Verb
ran           — Verb>Verb

Dynamic Filter Logic Words has the Filters Logic Words and Gerunds.

Dynamic Filter Verbs has the Filters Verbs and Gerunds.

The Description of runner can be Logic Word, Logic Word>Gerund or Gerund when the word runner is used as a Logic Word.

The Description of runner can be Verb, Verb>Gerund or Gerund when the word runner is a Verb Word.

The Description of runs, running and ran is Logic Word or Logic Word>Logic Word, for each word individually.

The Description of runs, running and ran can be Verb or Verb>Verb, for each word individually. Runs, running and ran cannot be a Noun or a Gerund.
